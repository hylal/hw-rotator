HW Rotator adds a whatsapp buttons on every single product page, that allow your costumer to buy through whatsapp.


== Description ==
HW Rotator help you to put two different Whatsapp number in every single product page alternately. 
You may also custumize the plugin css button on advance setting tab.

Demo [Click Here](site.com)






== Installation ==
1. You have to install Woocommerce before using this plugin 
2. Download HW Rotator Plugin
3. Upload the plugin files to your Wordpress
4. Activate the plugin
5. Go to Woocommerce Menu then click HwRotator
6. Input 2 Whatsapp Number (don't forget to put prefix your country code ex: 62 without +)
7. Save it
